# 🌸 She Can! Foundation — NGO Website

A modern, responsive website built for **She Can! Foundation**, a student-first NGO dedicated to empowering women through safe, rejection-free learning environments, free tech bootcamps, direct mentorship, and leadership growth programs.

🔗 **Live Site:** [ngo-website-roan-sigma.vercel.app](https://ngo-website-roan-sigma.vercel.app/)

---

## ✨ About the Project

This website showcases the NGO's mission, programs, events, volunteer opportunities, and donation initiatives — built to help the organization reach more students, volunteers, and donors online with a clean, modern, and mobile-friendly experience.

## 📦 Installation & Local Setup

1. **Clone the repository**
```bash
   git clone https://github.com/shaktipy/NGO-website.git
   cd NGO-website
```

2. **Install dependencies**
```bash
   npm install
```

3. **Run the development server**
```bash
   npm run dev
```

4. Open the URL shown in the terminal (usually `http://localhost:5173`)
